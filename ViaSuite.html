The ViaVersion suite allows server owners as well as players to allow client connection to servers of different Minecraft versions through a bit of magic™️. Below, you can find an overview of the projects to find the one that suits your exact needs.

Paper, Spigot, Velocity, Bungee & Sponge plugins

ViaVersion
.

Allows all client versions that are newer than your server

Get ViaVersion
  • Additional benefits: Less gameplay and anti-cheat issues if run on a Paper server
Install on: Server 1.8.8-latest
ViaBackwards
Requires ViaVersion

Allow Client versions that are older than your server down to 1.9

Get ViaBackwards
Install on: Server 1.10-latest
ViaRewind
Requires ViaVersion

Allow 1.7.10-1.8.9 clients

Get ViaRewind
  • Additional benefits: Less gameplay issues if using ViaRewind Legacy Support
Install on: Server 1.8.8-latest

Fabric & Forge mods

ViaFabric

Connecting to 1.8.8-latest servers

Get ViaFabric
  • Additional benefits: Less potential for mod incompatibilities than ViaFabricPlus
Install on Fabric client or server: Latest minor version of 1.8, 1.14-latest
ViaFabricPlus

Connecting to classic c0.0.15 to the latest release servers

Get ViaFabricPlus
  • Additional benefits: Less gameplay and anti-cheat issues
Install on Fabric client: Latest
ViaForge

Connecting to 1.8.8-latest servers

Get ViaForge
  • Additional benefits: Less gameplay and anti-cheat issues
Install on Forge client: Latest minor version of 1.8, 1.14-latest

Proxies for clients

ViaAAS

Connecting to 1.8.8-latest servers

Get ViaAAS
  • Additional benefits: Has already running, public instances
Install on: Local or remote instances to any server
ViaProxy

Connecting to classic c0.0.15-latest servers

Get ViaProxy
  • Additional benefits: User-friendly graphical user interface for local instances
Install on: Local instances to any server, or remote instances to a specific server
